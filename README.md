# Groupfindr

Prototype of an application that facilitates forming groups in an interactive manner.


## Prepare

1. Install [Node.js](https://nodejs.org/) 0.12.x


## Build

```bash
npm install
```


## Run


```bash
npm start
```

Alternative (requires [foreman](https://github.com/ddollar/foreman)):

```bash
foreman start
```
